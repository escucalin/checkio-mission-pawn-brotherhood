Fast jeder auf der Welt kennt das uralte Spiel Schach und hat zumindest ein grundlegendes Verständnis der Regeln. Bei diesem Spiel gibt es 6 verschiedene Figuren, die sich nur nach bestimmten Zugregeln bewegen dürfen. Daraus ergibt sich eine weite Palette an möglichen Spielkonstellationen. (siehe auch: Number of possible chess games at the end of the n-th plies )
In dieser Aufgabe musst Du die Bewegungen und das Verhalten einer dieser Figuren unter die Lupe nehmen.

Schach ist ein Zweispielerstrategiespiel. Eine Schachpartie wird auf einem quadratischen Brett gespielt, das in 8 x 8 abwechselnd schwarze und weiße quadratische Felder eingeteilt ist. Die Zeilen werden als Reihen bezeichnet und mit den Zahlen 1 bis 8 beschschriftet, während die Spalten als Linien bezeichnet und mit den Buchstaben a bis h beschriftet werden. So lässt sich jedem Feld ein eindeutiges Koordinatenpaar zuordnen - ein Buchstabe und eine Zahl (z.B.: "a1", "h8", "d6").

Hier geht es ausschließlich um die Bauern.
Ein Bauer kann eine gegnerische Figur auf einem Feld schlagen, das diagonal ein Feld vor ihm liegt. Im Fall eines weißen Bauern haben die vor ihm liegenden Felder eine höhere Reihennummer.
Ein Bauer ist in der Regel eine schwache Figur, aber es gibt acht von ihnen und somit kann man eine Bauernkette als Verteidigung errichten.
Bei dieser Strategie verteidigen die Bauern in den hinteren Reihen die Bauern vor ihnen. Ein Bauer gilt als gedeckt, wenn ein anderer Bauer der gleichen Farbe, eine gegnerische Figur auf dessen Feld schlagen könnte.
Es befinden sich mehrere weiße Bauern auf dem Schachbrett - und zwar nur weiße Bauern!

Gegeben ist ein Set mit den Koordinaten der Felder, auf denen ein weißer Bauer platziert ist. Du sollst herausfinden, wie viele dieser Bauern gedeckt sind.

Eingabe: Die Koordinaten der weißen Bauern als Set von Strings.

Ausgabe: Die Anzahl der gedeckten Bauern als ganze Zahl.
